A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas Web. Documentos HTML podem ser interpretados por navegadores.

Até este momento, vimos os seguintes recursos da linguagem:

Recurso Tags associadas Descrição
links <a> Criar links entre páginas
listas <ol> e <li> Criar listas ordenadas
<ul> e <li> Criar listas não ordenadas
Imagens <img> Inserir imagens
